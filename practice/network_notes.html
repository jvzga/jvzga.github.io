Today's Networking Notes

IP Protocols

  • Multiple numbers of packets depending on file/data size
  • Each packet has a header with source and destination IP addresses
  • Packets may take different routes to the destination
  • Packets are reassembled at the destination
  • The internet is highly connected with various protocols
  • Some protocols are TCP, UDP, IP, HTTP, FTP, SMTP, etc.
  • TCP/IP is the practical model while the OSI model is the theortical model
  • The constantly gets mapped to have a updated network topography

Networking

  • 1% of data gets lost in the process
  • Packets have sequence numbers to help reassemble them in the correct order
  • Checks are used to verify data integrity
  • Retransmission of lost packets is requested by the receiver
  • Control mechanisms prevent network congestion
  • Routers direct packets based on IP addresses
  • Switches operate at the data link layer to forward info
  • Firewalls monitor and control incoming and outgoing network traffic
  • VPNs provide secure remote access to a network
