HOW TO USE THE VIM EDITOR USING THE TERMINAL

To open or make a file in vim, you can follow these steps:

  • Navigate to directory/area where you will create your file or will edit an file
  • Type in vim (newfilename) or vim (existingfilename)
  • Press 'Enter'
  • Congrats! you are now in the vim editor

To exit vim, you can follow these steps:

  1. Press the 'Esc' key to ensure you are in normal mode.
  2. Type ':q' and press 'Enter' to quit if you haven't made any changes.
  3. If you have made changes and want to save them before quitting, type ':wq' and press 'Enter'.
  4. If you want to quit without saving changes, type ':q!' and press 'Enter'.
