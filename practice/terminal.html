HOW TO USE THE VIM EDITOR USING THE TERMINAL

To open or make a file in vim, you can follow these steps:

  1. Navigate to directory/area where you will create your file or will edit an file
  2. Type in vim (newfilename) or vim (existingfilename)
  3. Press 'Enter'
  4. Congrats! you are now in the vim editor
  5. To start editing the file, press the 'i' key to enter insert mode
  6. Make your changes or add your content to the file

Lets say you made the following file below and you want to know how to save/quit the file

		This is some preformatted text.
		It preserves spaces and line breaks.		
		And even indentation.

To exit vim, you can follow these steps:

Note:There are different ways to exit vim depending on whether you want to save changes or not.

  • Press the 'Esc' key to ensure you are in normal mode.
  • Type ':q' and press 'Enter' to quit if you haven't made any changes.
  • If you have made changes and want to save them before quitting, type ':wq' and press 'Enter'.
  • If you want to quit without saving changes, type ':q!' and press 'Enter'.
